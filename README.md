# Wedding Website

A static wedding website built with HTML, CSS, and JavaScript, backed by
a decoupled C# Web API for uploads, media management, and services.

## Architecture
- Frontend: GitHub Pages (HTML/CSS/JS)
- Backend: ASP.NET Core Web API (Azure App Service)
- Media Uploads: Google Drive 
- Data Storage: Supabase (registry, guests, logs)
